## 练习结构

Week 1 Exercises 包含两个交互测评：

1. **Week 1 Exercises: Hyperplanes**
   - Vertical block: `week1_exercises_1_vert`
   - LTI block: `Week_1_Exercises_Hyperplanes`

2. **Week 1 Exercises: numpy**
   - Vertical block: `week1_exercises_2_vert`
   - LTI block: `Week_1_Exercises_numpy`

- `week1_exercises_public_page.html`：公开课程页面的完整 HTML。

## 练习内容

完整练习包含：

- Hyperplanes
  - Through origin
  - General hyperplane and distance to origin
- Numpy intro
  - Array
  - Transpose
  - Shapes
  - Row and column vectors
  - Vector length and normalization
  - Indexing
  - Representing data
  - Matrix multiplication
